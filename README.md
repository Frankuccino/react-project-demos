# React Learning Lab: 10-Project Series (Beginner)

This repository is a curated collection of React applications designed to master the fundamentals of modern frontend development. What started as a 9-project series evolved into a study of **modular architecture**, **scoped styling**, and **single-page navigation**.

---

## 🚀 The Learning Journey

This series documents the transition from "learning syntax" to "building scalable systems." After completing the initial builds, the entire repository was **refactored for prestige**:

- **Architectural Overhaul:** Transitioned from global CSS to **CSS Modules** to prevent style leakage.
- **Routing System:** Implemented **React Router v6** to unify independent projects into a seamless dashboard experience.
- **Component Logic:** Mastered the "Page Wrapper" pattern to handle unique project themes (Dark/Light/Gradients) within a single SPA.

### 🛠 Core Concepts Explored

- **Scoped Styling:** Implementing `*.module.css` for true component encapsulation.
- **State & Props:** Mastering one-way data flow and lifting state to parent containers.
- **Hooks Deep-Dive:** Heavy use of `useState`, `useEffect`, and `useRef`.
- **Navigation:** Managing browser history and dynamic routing via `Link` and `Routes`.
