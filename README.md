# React Learning Lab: 9-Project Series

This repository is a collection of 9 distinct React projects designed to master the fundamentals of modern frontend development. Each project focuses on a specific core concept, progressing from basic UI rendering to dynamic, state-driven applications.

---

## 🚀 The Learning Journey

This series documents my transition from "learning syntax" to "building systems." By keeping the logic inside "Vanilla React," I focused on mastering the underlying mechanics of the library before moving to external state management tools.

### 🛠 Core Concepts Explored

- **Component Architecture:** Breaking down complex UIs into small, reusable pieces.
- **State & Props:** Mastering the one-way data flow and lifting state to parent components.
- **Hooks Deep-Dive:** Heavy use of `useState`, `useEffect`, and `useId`.
- **Event Handling:** Managing user interactions across buttons, inputs, and forms.
- **Dynamic Rendering:** Mapping through data arrays to generate performant lists.
